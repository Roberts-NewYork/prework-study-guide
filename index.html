CSS

  • CSS layout is mostly based on the Box Model.
  • Padding is the space around the content.
  • Border is the solid line that is just outside the padding.
  • Margin is the space around the outside of the border.
  • The elements of a CSS file are called a "ruleset" made up of a selector and a declaration. The selector is called a rule.
  • Multiple selectors can apply to one declaration.
  • There must be a colon (:) between the selector and declaration, commas separate multiple selectors.
  • Declarations are separated by a semicolon (;).

GIT

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it.
  • git add -A adds all files in the folder.
  • git commit -m: a message that describes what the git contains.
  • git push origin: pushes all code to GitHub for a Pull Request.
  • git pull origin main: syncs the local branch with the base branch in GitHub.
