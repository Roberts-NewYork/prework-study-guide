HTML

CSS

  • CSS layout is mostly based on the Box Model.
  • Padding is the space around the content.
  • Border is the solid line that is just outside the padding.
  • Margin is the space around the outside of the border.
  • Text-shadow sets a drop shadow on the text inside the element.
  • There is the selector and the declaration, which contains the property and it's values which are separated by a colon (:), multiple declarations are separated by semicolon.
  • There can be multiple selectors and multiple rules separated by semicolons (;)
  • Each ruleset must be wrapped in curly braces ({}) excluding the selector.

GIT

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A adds all files in the folder.
  • git push origin main: pushes all code to GitHub for a Pull Request.
  • git commit -m: a message that describes what the git contains.
